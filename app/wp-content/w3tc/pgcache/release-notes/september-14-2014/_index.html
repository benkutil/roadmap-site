Basecamp Browser Extension

The Basecamp (New or Classic) extension allows you to manage start dates, estimates and track time from within Basecamp. The extension is only available for Google Chrome initially with other browsers trailing by a few weeks. View related support article.

Click image to view larger version

 Manage Start Dates & Estimates

Add or edit start dates in Basecamp. Changes made in the extension update Roadmap immediately.
Click image to view larger version

 Track Time

Use the timer or log time for a specific day.

To-Do Enhancements

Click image to view larger version

 To-Do Dependencies

To-do items and to-do lists support dependent relationships. Previously, the only way to set-up dependent relationships was to associate to-do lists with milestones, which are automatically converted into events in Basecamp (New). With to-do dependencies, milestones are optional.
Click image to view larger version

 Assign Multiple Resources to To-Dos

Assign multiple resources to a to-do item. Basecamp users: Roadmap will export the first assigned resource -- the responsible party -- to Basecamp.

Embed Tables & Images in Dashboard Widgets

Click image to view larger version

 Embed tables and and images in Free Text widgets. To embed images:
  • Paste the URL for the image; or
  • For attachments (files) uploaded to Roadmap, right-click the file name and copy the link. Important: Users who do not have access to the project associated with the file will not not be able to view it.

Public API v1.1

  • Related: Admins can enable or disable the API for the entire account. Once enabled, users can access their Roadmap identity token
  • Update project status + health, dates, and work items (including resources assigned to work items)
  • Delete work items
  • Read and update project attributes

Complete API documentation can be found on GitHub.

Miscellaneous

  • Resource 360 > Time. Actual column is now cumulative for the same work item, and the Projected column calculation is modified to account for the cumulative figures. Also, the Excel export now includes Actual and Projected figures.
  • Dashboard Utilization and Availability widgets. The sorting logic is changed to make the colors more consistent across projects.
  • Custom date range. Maximum timeframe expanded to allow up to 1 year.
